Privacy Policy

Tulip Field Generator

Last updated: 2026-04-06

Tulip Field Generator is a Chrome extension that visually transforms the currently active page into a stylised tulip-field overlay.

Data Collection

Tulip Field Generator does not collect, sell or share personal data.

Tulip Field Generator does not:

  • create user accounts
  • send page content to a remote server
  • use analytics, tracking pixels, or advertising SDKs
  • store browsing history outside the current browser session

What the Extension Accesses

Tulip Field Generator uses Chrome's activeTab and scripting permissions to run only when the user clicks the extension on the current tab.

When activated, the extension reads the visible structure of the current page in order to detect layout blocks, build local field territories, and render a visual overlay in the current tab.

This processing happens locally in the browser.

Data Storage

Tulip Field Generator does not store personal data on remote servers.

The extension does not maintain a backend service or external database.

Third-Party Sharing

Tulip Field Generator does not transmit user data to third parties.

Contact

For support or privacy questions, use the GitHub repository issue tracker or the support contact listed in the Chrome Web Store page.
